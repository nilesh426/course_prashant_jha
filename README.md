# course_prashant_jha

A beginner-focused Python practice repository. Scripts are organized by day and each file demonstrates a specific concept or short exercise (inputs, conditionals, loops, strings, lists, tuples, and small algorithms).

**Requirements**
1. Python 3.x

**How To Run**
1. Open a terminal at the repo root `d:\course`.
1. Run any file with:
```bash
python course_prashant_jha/DAY2/reverse.py
```

**Repository Structure**
1. `course_prashant_jha/DAY1` - basics, input/output, arithmetic, operators, and conditionals.
1. `course_prashant_jha/DAY2` - strings, loops, list/tuple operations, and practice tasks.
1. `course_prashant_jha/DAY3` - dictionaries, functions, arguments, nested loops, and string strip methods.
1. `course_prashant_jha/DAY4` - exception handling, predefined string methods, and practice tasks.
1. `course_prashant_jha/DAY5` - object-oriented programming concepts and practice exercises.

**DAY2 Detailed Index**

   Demonstrates `format()` with positional/keyword arguments and an f-string total.

   Indexing and slicing examples, including reverse slicing.

   String casing methods: `lower`, `upper`, `swapcase`, `title`, `capitalize`.

   Basic `for` loops with `range` and multiplication table patterns (commented).
1. `course_prashant_jha/DAY2/loop1.py`  
   Example for removing duplicate characters from a string (commented).
1. `course_prashant_jha/DAY2/reverse.py`  
   Inputs a string, reverses it, and checks palindrome status.
1. `course_prashant_jha/DAY2/tables.py`  
   Prints multiplication tables from 2-10 and 11-20 using loops.
1. `course_prashant_jha/DAY2/List.py`  
   Extensive list operation examples: indexing, slicing, updates, append/insert/remove, copy,
   multi-dimensional lists, multiplication/concatenation, clearing, list conversion from string,
   reverse/sort, aliasing, and a few small array exercises. The active code prints a slice

   Tuple creation, immutability notes, tuple concatenation, repetition, nesting, and length examples (commented).

   A collection of small practice tasks (most commented) including:

   sum of adjacent distances, break/continue examples, `zip()` usage,
   string cleanup with `*` separators, BODMAS, identity/equality checks,
   anagram check, and word counting. The active code builds a list of pairwise products
   for all unequal index pairs in `[1, 2, 3, 4, 5]`.
